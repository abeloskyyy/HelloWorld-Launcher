Frequently Asked Questions

Everything you need to know.

Why does it say it has a virus?

Because the app is not signed (certificates cost money). It's a common false positive in open source apps.

Is it legal?

Yes. The launcher uses a legal copy of the game according to Mojang/Microsoft terms.

Does it support large mods?

Yes, it supports heavy modpacks as long as your PC has enough resources.

Does it work without internet?

Yes, you can use Offline Mode to play versions already downloaded.

Does it support Fabric and Forge?

Absolutely. You can create independent profiles for each.
